Setup Login Dashboard Students Books Borrow Return Activity Logs Reports User Management Tips

Setup

  • Install XAMPP and start MySQL.
  • Import PublicSchoolLibrarySystem/Database/library_db.sql into MySQL.
  • Build: dotnet build PublicSchoolLibrarySystem.sln
  • Run: open the solution in Visual Studio and press F5, or run the compiled app.
  • Default users:
    • Admin admin / admin123
    • Librarian librarian / lib123

Login Flow

1) Open App

  • Start at the Login screen.
  • Select a role and enter credentials.

2) Successful Login

  • App navigates to the Dashboard.
  • Top-right shows your user info.

Dashboard

  • Left sidebar: navigation to Books, Students, Borrow/Return, Activity Logs, Reports, Users.
  • Main tiles summarize counts: books, students, borrowed, returned.
  • Every page has a Back button to return to Dashboard.

Manage Students

Add / Update

  • Enter Full Name.
  • Select College: CICS, CTED, CHM, CA, CCJE, CBEA.
  • Select Year (1â€“4) and Section (Aâ€“B).
  • Optional: contact number.
  • Click Add or Update.

Delete

  • Select a student in the grid and click Delete.
  • If the student has borrow history, youâ€™ll be prompted to remove those records; active borrows will restore book stock.

Manage Books

  • Add or update book details: title, author, category, ISBN, quantity.
  • Delete books that are no longer in the catalog.

Borrow Flow

  1. Open Borrow / Return.
  2. Select a Student and a Book from dropdowns.
  3. Click Borrow; quantity decreases by one.
  4. Borrowed records appear in the grid with status Borrowed.

Return Flow

  1. Select a row in the borrowed grid.
  2. Click Return; status becomes Returned, quantity increases by one.

Activity Logs

  • Shows actions like Add/Update/Delete across modules.
  • Useful for auditing changes.

Reports

  • Summaries of borrowed/returned activity and counts.
  • Use filters (if available) to refine results.

User Management

  • Add and update users with roles: Admin or Librarian.
  • Delete users when necessary.

Tips

  • Use the Back button on each page to return to Dashboard.
  • Students grid headers: College, Year & Section.
  • Database connection string is in Modules/DbConnection.vb.
  • Seed books are auto-imported at startup; see Modules/DatabaseSeeder.vb and Modules/BookImporter.vb.
